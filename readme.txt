Job Blitz is easy to use.
Simply add [job-blitz] tag wherever you would like the feed to display.

== Description ==
This feed converts your html comment tag into an iframe and loads information from job-blitz.com.

You may want to set the height and width of the feed example:

[job-blitz height=500 width=300]

== Installation ==


1. Upload `job-blitz-feed.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[job-blitz]` in your templates, posts or pages
